# Agrobot
Our project addresses the lack of accessibility in agricultural spaces such as community gardens, vertical farms, and composting sites, which are rarely designed with ADA compliance in mind. We propose an autonomous robot that can be deployed in two complementary ways. First, it can be used as an accessibility assessment tool to traverse a site both in advance and for real-time feedback. It can identify whether pathways, turning radii, surface conditions, and clearances are navigable, based on the principle that if a robot can reliably traverse a space, a person with ADA accessibility needs likely can as well. Second, the robot can operate as an autonomous follower that accompanies a person with ADA accessibility needs through these environments, providing navigation assistance and on-site support. By using the robot as both an ADA testing platform and as an assistive mobility companion, this project aims to make community gardens and similar spaces more accessible to people with disabilities while establishing a flexible foundation for future extensions.
